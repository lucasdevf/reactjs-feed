# FEATURES
1. See posts in feed
2. Add comments for each post
3. Remove added comment
4. Add like to post

<a id="howToRun"></a>
# HOW TO RUN
1. Clone repository
2. Run ```yarn install``` to install dependencies
3. Run ```yarn dev```
